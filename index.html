Reference table for CSS properties.

Property Description
Width The width of content in a box model can be used with numerical values like px or % Example: width: 10px
Height The height of content in a box model can be used with numerical values like px or % Example: height: 10px
Padding The amount of space between content area and the border of an element.
border The thickness and style of the border surrounding the content area and the padding.
border-radius Borders are naturally a square around an element or well better said, a box, with border radius they don't have to be though. This property modifies the corners of an elements border using pixels
margin The amount of space between the border and the outside edge of the content.
Background-color Changes the background color of and element
overflow When content overflows or spills outside of it's box model we can use the overflow property along with it's values hidden, scroll or visible
visibility The visibility property allows us to hide elements from view by using the values hidden, visible, or collapse
